## [ChatApp]

A chat application built using socket.IO on Node.JS framework

### [Setup]

Download the application by running this command *git clone https://github.com/ayushanand29/chatApp*

cd inside the folder
Use these dependencies installation as :

> npm i --save-dependency socket.io

> npm i --save-dependency nodemon

### [Running]

To run the project, simply execute this command while being present in the project folder :

> npm start
